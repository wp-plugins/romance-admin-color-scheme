==Installation== 

1. Download Romance Admin Color Scheme package file, extract romance-admin-color-scheme directory from ZIP file, upload it to WordPress Plugins directory '/wp-content/plugins/' or install plugin from Plugin panel in WordPress Dashboard.

2. Activate Romance Admin Color Scheme via Plugin page.                 

3. Under Users > Your Profile > Admin Color Scheme, select Romance and Save.

If you are new to WordPress : [Installing Plugins](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins)
